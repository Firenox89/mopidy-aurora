Installation
============

Install by running::

    pip install Mopidy-Aurora

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Configuration
=============

Mopidy-Aurora to your Mopidy configuration file::
